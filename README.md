# 📝 开发学习笔记

> 有条理地记录学习开发过程中遇到的问题、解决方法和感悟。

## 项目结构

```
dev-learning-notes/
├── README.md            ← 你正在看的这个文件
├── 笔记模板.md           ← 每次写笔记复制这个模板
├── notes/               ← 按日期记录的笔记（日志式）
│   └── 2026-07/         ← 按年月分文件夹
│       └── 2026-07-15-标题.md
├── topics/              ← 按主题整理的深度笔记
│   └── 例如-Python异步编程.md
└── assets/              ← 笔记中引用的图片、截图
```

## 两种笔记模式

| 模式 | 用途 | 命名规范 |
|------|------|----------|
| **日志笔记** (notes/) | 每天遇到的问题、随手记录 | `YYYY-MM-DD-简短标题.md` |
| **主题笔记** (topics/) | 某个主题的系统总结 | `主题名.md` |

## 怎么写一篇笔记

1. 复制 `笔记模板.md` 的内容
2. 在 `notes/年-月/` 下新建文件，命名为 `2026-07-15-解决了什么问题.md`
3. 填写内容
4. 保存后用 Git 上传（见下方）

## 怎么上传到 GitHub

```bash
# 1. 进入项目目录
cd D:\dev-learning-notes

# 2. 添加所有改动
git add .

# 3. 提交，写一句话说明今天做了什么
git commit -m "记录：解决XXX问题"

# 4. 推送到 GitHub
git push
```

就这四步，每次写完笔记执行一遍即可。
